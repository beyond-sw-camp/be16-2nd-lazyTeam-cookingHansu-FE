## 🛠️ 기술 스택

### 📦 Frontend

- **Vue 3**
- **Vite**
- **Vuetify 3**
- **Pinia** – 상태 관리
- **Vue Router 4** – SPA 라우팅
- **Axios** – API 통신
- **SCSS** – 스타일링
- **JavaScript (ES6+)**

🎯 Commit Convention
---

| 태그         | 설명                   |
| ---------- | -------------------- |
| `feat`     | 기능 추가                |
| `fix`      | 버그 수정                |
| `refactor` | 리팩토링 (기능 변경 없음)      |
| `style`    | 코드 스타일 수정 (공백, 포맷 등) |
| `docs`     | 문서 작성/수정             |
| `test`     | 테스트 코드 작성            |
| `chore`    | 빌드, 패키지, 설정 등 기타 변경  |
| `ci`       | CI/CD 설정 관련          |
